# Introduction 
A simple, educational honeypot written in Python with a graphical user interface (GUI). This project demonstrates how honeypots work by listening on a specified port, logging connection attempts, and displaying activity in real-time.

## Project structure 
''' text
Honeypot /
├── honeypot_logger.py       # Logger configuration (centralized logging)
├── honeypot_functions.py    # Core honeypot logic (socket handling)
├── honeypot_gui.py          # Tkinter GUI for control and monitoring
├── honeypot_main.py         # Entry point (launches the application)
└── README.md                # This file
'''
